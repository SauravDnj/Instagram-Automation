# Instagram Automation

🚀 **100% FREE Instagram Automation Tool** - Post photos, videos, reels, and stories automatically!

![Version](https://img.shields.io/badge/version-2.0.0-blue)
![Python](https://img.shields.io/badge/python-3.8+-green)
![License](https://img.shields.io/badge/license-MIT-orange)

## ✨ Features

- ✅ **Post Photos** - Upload images to your Instagram feed
- ✅ **Post Videos/Reels** - Share video content and reels
- ✅ **Post Stories** - Photo and video stories
- ✅ **Add Captions** - Full caption and hashtag support
- ✅ **Schedule Posts** - Schedule for any future date/time ⭐ NEW!
- ✅ **Recurring Schedules** - Daily, weekly, monthly, yearly auto-posting ⭐ NEW!
- ✅ **Manage Schedules** - View and cancel scheduled posts ⭐ NEW!
- ✅ **100% FREE** - No API keys or subscriptions required!
- 🔜 **Coming Soon**: Analytics, image editing, multiple accounts

## 🎯 No API Keys Required!

Unlike other Instagram tools, this automation works with **just your Instagram username and password**. No paid APIs, no monthly subscriptions!

## 📦 Installation (Desktop App)

### Prerequisites
- Windows 10/11
- Python 3.8 or higher

### Quick Start

1. **Clone the repository**
```bash
git clone https://github.com/SauravDnj/Instagram-Automation.git
cd Instagram-Automation
```

2. **Create virtual environment**
```bash
python -m venv venv
venv\Scripts\activate
```

3. **Install dependencies**
```bash
pip install -r requirements.txt
```

4. **Run the application**
```bash
python main.py
```

## 🖥️ Usage

### Quick Post (Instant)
1. **Login** - Enter your Instagram username and password
2. **Choose Post Type** - Photo, Video/Reel, or Story
3. **Select Media** - Pick your image or video file
4. **Add Caption** - Optional caption with hashtags
5. **Post!** - Instantly publish to Instagram

### Schedule Posts ⭐ NEW!
1. **Click "Schedule Post for Later"** button
2. **Choose Schedule Type**:
   - **One-Time**: Select specific date and time
   - **Recurring**: Daily, weekly, monthly, or yearly
3. **Select Media** - Choose file or folder (for recurring)
4. **Add Caption** - Write your caption
5. **Schedule!** - Posts automatically at scheduled time

### Manage Scheduled Posts
- View all scheduled posts in the "Scheduled Posts" tab
- Cancel any scheduled post
- See post status and timing
- Auto-refresh every 30 seconds

**📖 Full scheduling guide**: See [SCHEDULING_GUIDE.md](SCHEDULING_GUIDE.md)
